What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents, representing the HTML/XML structure as a tree of objects, allowing manipulation via scripts like JavaScript.

How do you select an element from the DOM?

Use methods like getElementById(), querySelector(), getElementsByClassName(), or getElementsByTagName() to select elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows attaching one event listener to a parent element to handle events from its child elements, improving performance and handling dynamic content.

How do you manipulate an element's attributes and styles using the DOM?

Use methods like setAttribute(), getAttribute(), element.style, classList.add(), and classList.remove() to change attributes or styles.
